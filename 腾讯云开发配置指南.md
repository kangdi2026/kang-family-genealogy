## 🔧 第二步：创建云开发环境

### 1. 点击"新建"按钮
进入控制台后，点击"新建环境"或"立即创建"

### 2. 填写环境信息

```
环境名称: kang-genealogy
备注: 康氏家族族谱
计费方式: 按量计费（免费额度）⭐ 重要！选这个
环境类型: Web 应用
```

### 3. 点击"立即开通"
等待环境创建完成（约 30 秒 - 1 分钟）

---

## 💾 第三步：开通数据库

### 1. 进入数据库管理
- 左侧菜单找到"数据库"
- 点击进入

### 2. 开通云数据库
如果是第一次使用，需要点击"开通"按钮

### 3. 创建集合（数据表）

点击"添加集合"：
```
集合名称: family_intro
```

### 4. 设置数据库权限

点击刚创建的 `family_intro` 集合，进入权限设置：

**权限设置**：
```
读权限: 所有用户可读
写权限: 所有用户可写（后面我们用密码保护）
```

或者点击"自定义安全规则"：
```json
{
  "read": true,
  "write": true
}
```

---

## 🔑 第四步：配置 Web 安全域名

### 1. 进入设置
左侧菜单点击"设置" → "安全配置"

### 2. 添加 Web 安全域名

在"Web 安全域名"栏点击"添加域名"：

```
kangdi2026.github.io
```

**重要**：不要添加 `https://` 前缀，只填域名！

如果想本地测试，还可以添加：
```
localhost
127.0.0.1
```

### 3. 保存设置

---

## 📋 第五步：获取环境配置信息

### 1. 进入环境设置
左侧菜单点击"设置" → "环境设置"

### 2. 查看环境信息

你会看到：
```

环境 ID: kang-genealogy-8gxxxxx（重要！）
创建时间: 2026-xx-xx
```

### 3. 复制环境 ID

**环境 ID 格式示例**：
```

```

### 4. 获取初始化信息

向下滚动，找到"应用凭证"或"初始化配置"：

会看到类似这样的信息：
```
环境 ID (envId): kang-genealogy-8g12345
区域 (region): ap-shanghai（或其他）
```

## 🆘 常见问题

### Q1: 找不到"数据库"菜单？

A: 选择"按量计费"并使用免费额度，完全免费

A: 创建时默认分配，一般是上海或广州

A: 确保只填域名，不要加 https:// 或尾部斜杠
