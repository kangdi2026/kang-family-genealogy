# GitHub Pages 部署详细指南

## 📌 完整部署流程

### 第一步：注册 GitHub 账号

1. **访问 GitHub 官网**
   - 打开浏览器，访问：https://github.com

2. **注册新账号**
   - 点击右上角的 **Sign up**（注册）按钮
   - 填写信息：
     * **Email**：输入你的邮箱（建议用常用邮箱）
     * **Password**：设置密码（至少8位，包含字母和数字）
     * **Username**：设置用户名（这个很重要，会出现在网址中）
       - 例如：`kangdi123`
       - 最终网址会是：`https://kangdi123.github.io/kang-family-genealogy/`

3. **验证邮箱**
   - GitHub 会发送验证邮件到你的邮箱
   - 打开邮箱，点击验证链接
   - 完成验证后登录 GitHub

---

### 第二步：创建新仓库

1. **点击创建仓库**
   - 登录后，点击右上角的 **+** 号
   - 选择 **New repository**（新建仓库）

2. **填写仓库信息**
   ```
   Repository name（仓库名）: kang-family-genealogy

   Description（描述）: 康氏家族族谱

   Public/Private: 选择 Public（公开）

   ⚠️ 重要：不要勾选以下选项
   □ Add a README file
   □ Add .gitignore
   □ Choose a license
   ```

3. **创建仓库**
   - 点击绿色按钮 **Create repository**

---

### 第三步：记录你的用户名

创建完仓库后，GitHub 会显示一个页面。请记录以下信息：

- **你的 GitHub 用户名**：________________
- **仓库地址**：会显示为 `https://github.com/你的用户名/kang-family-genealogy`

---

### 第四步：执行推送命令

**请将你的 GitHub 用户名告诉我，我会生成具体的命令**

命令格式示例（假设用户名是 `kangdi123`）：
```bash
cd /c/Users/kangdi1/genealogy-web
git remote add origin https://github.com/kangdi123/kang-family-genealogy.git
git branch -M main
git push -u origin main
```

执行时会要求输入 GitHub 账号和密码（或 token）

---

### 第五步：启用 GitHub Pages

1. **进入仓库设置**
   - 在你的仓库页面，点击顶部的 **Settings**（设置）

2. **找到 Pages 设置**
   - 在左侧菜单中，找到并点击 **Pages**

3. **配置 Pages**
   ```
   Source:
   - Branch: main
   - Folder: / (root)
   ```
   - 点击 **Save**（保存）

4. **等待部署**
   - 大约 1-3 分钟后，页面会显示网址
   - 网址格式：`https://你的用户名.github.io/kang-family-genealogy/`

---

## 🎉 完成！

部署成功后，你会得到一个在线网址，可以分享给家人访问。

**你的族谱网址**：`https://你的用户名.github.io/kang-family-genealogy/`

## ⚠️ 常见问题

### Q1: 推送代码时要求输入密码，但密码不对？
**A:** GitHub 现在不支持密码登录，需要使用 Personal Access Token（个人访问令牌）

**解决方法**：
1. 访问：https://github.com/settings/tokens
2. 点击 **Generate new token** → **Generate new token (classic)**
3. 填写信息：
   - Note: `genealogy-web-deploy`
   - Expiration: 选择 **No expiration**（不过期）
   - 勾选权限：**repo**（全选）
4. 点击 **Generate token**
5. **复制生成的 token**（只显示一次，务必保存）
6. 推送代码时，用 token 代替密码

### Q2: 网页显示 404 错误？
**A:** 请检查：
1. GitHub Pages 设置中的 Branch 是否选择了 **main**
2. 等待 3-5 分钟让 GitHub 完成部署
3. 刷新页面试试

### Q3: 想修改网站内容怎么办？
**A:** 修改本地文件后，执行：
```bash
cd /c/Users/kangdi1/genealogy-web
git add .
git commit -m "更新族谱内容"
git push
```
